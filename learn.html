How Internet Speed Tests Work (HTML5 vs Flash)

An internet speed test online downloads random data to your browser, measures time and converts it to Mbps. Modern HTML5 speed test tools use fetch() and ReadableStream – no Flash, no Java.

Step-by-step process (Cloudflare edge example)

  1. Your browser requests __down?bytes=10000000 (10 MB).
  2. Server streams data over HTTPS (HTTP/3 + BBR if available).
  3. Client measures bytes per millisecond.
  4. We discard the slowest 10 % (TCP slow-start) and average the rest.
  5. Result shown in Mbps and on the live gauge.

Why 10 MB sample size?

A 1 MB sample finishes too quickly on fibre and gives inflated bursts. Ten megabytes survives TCP slow-start yet finishes in < 3 s on 100 Mbps lines, so you see **sustainable, real-world figures** rather than marketing peaks.

Speed Test Accuracy – Why Results Vary

Speed test accuracy depends on WiFi interference, device limits, network congestion, server peering and time of day. Run 3 tests and average for reliable figures.

  • WiFi vs Ethernet: cable eliminates radio noise.
  • Background apps: pause cloud backups, Windows updates.
  • Server distance: pick edge < 50 km (we auto-do).
  • Time of day: peak hours (7-11 pm) show worst-case.

WiFi Speed Test vs Ethernet – Which to Trust?

Factors That Affect Speed (WiFi, ISP, Device)

  • Wi-Fi signal: distance, walls, interference (microwaves, baby monitors)
  • Device limits: old Wi-Fi 4 chips cap at ~50 Mbps
  • ISP peering: some providers throttle streaming ports in evening hours
  • Background apps: cloud backups, Windows updates, Dropbox sync
  • Ethernet vs Wi-Fi: cable eliminates radio interference

Speed Test for Gaming – Good Ping & Jitter

For speed test for gaming you need < 30 ms ping, < 5 ms jitter and 0 % packet-loss. Use Ethernet, close background apps and pick nearest server.

Recommended gaming speeds (2025)

  • Console (PS5/Xbox): 50 Mbps down / 10 Mbps up / < 30 ms ping
  • PC competitive (Valorant, CS2): 100 Mbps down / 20 Mbps up / < 15 ms ping / < 3 ms jitter
  • Cloud gaming (GeForce NOW): 25 Mbps down / < 20 ms ping / < 5 ms jitter

How to Improve Internet Speed – 6 Proven Steps

  1. Re-start modem/router – clears memory leaks.
  2. Update firmware – security + performance patches.
  3. Switch Wi-Fi channel – use 1, 6 or 11 on 2.4 GHz; auto on 5 GHz.
  4. Use 5 GHz or 6 GHz – less congested, higher throughput (shorter range).
  5. Limit background apps – pause cloud backups during calls.
  6. Upgrade plan – fibre > cable > DSL > satellite.
