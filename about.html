About Us

We built the speed test we always wanted: private, ad-free and open-source.

Privacy First

Zero tracking, zero ads. Your IP and test results never leave your browser. We do not set cookies, log addresses or share data with third parties—ever.

Open Source

MIT-licensed code. Fork, modify or embed freely. Issues and pull requests are welcome on the repository.

Netlify Powered

Hosted on Netlify’s global CDN for sub-second load times and perfect Core Web Vitals. The site is served over HTTPS with automatic deploys from Git.

Accurate Results

We use Cloudflare’s 10 MB edge endpoint and real-time browser timers to mirror everyday download conditions. Measurements finish in under three seconds on most broadband lines.

Why We Built This

Most speed tests are bloated with ads, tracking scripts and pop-ups that skew the very metric they claim to measure. We wanted a tool that loads instantly, runs entirely in the browser, and disappears the moment you have your number. By hosting on Netlify and using Cloudflare’s public endpoints we eliminate backend costs while keeping tests fast, repeatable and transparent. The code is small enough to audit in minutes, and the UI is dark-by-default to save energy on OLED screens. Whether you are debugging Wi-Fi, comparing ISPs or simply curious, you can run the test as often as you like without creating an account or accepting cookies. Accuracy comes from sampling real HTTPS traffic, not from synthetic plugins or Flash applets. We hope the project helps users, developers and network engineers alike to get honest, privacy-respecting speed data in the shortest time possible.
